Import Files[edit]

Import Bitmap Files[edit]

The laser cutter can raster images from bitmap files.

  • Laser Cut imports the following bitmap file formats
    • BMP, JPG, GIF, PNG, TIF, TGA, PCX

The software can only set engrave (raster) cuts for bitmaps; it needs vector drawings to set other cut types.

Import Vector Files[edit]

Use vector files to set cut lines and mark engrave areas.

  • Laser Cut imports following vector file formats
    • DXF (AutoCAD)
    • AI (Adobe Illustrator 10 preferred)
    • PLT (Plotter file)
    • TST, NC
    • ECP files -- this is the native format of Laser Cut

By default the software automatically selects all embedded vectors on file import.

Associate Vectors Sets to Color Layers[edit]

  • If you want to order your cuts or make more than one type of cut, you need to put your vectors into layers
    • Select all the vectors for a given layer
    • Select a color for the layer
    • Repeat selecting vectors and defining layers until all vectors are within a layer

Set Layer Options[edit]

  • Use the drop-down menu next to the layer color to select the type of laser cut (cut, engrave, grade engrave, or hole)
  • Double-click the layer color to bring up the options or the cut type selected
  • Use the materials chart on the wall as a guide when setting option values
  • If not selected, make sure to select the Always Blow radio button
  • To keep a layer active but take no laser action on it, deselect its output box.


The large laser does not accept any cut speed slower than 5. If you set a slower cut speed, during download the software kicks an unspecified communication error. Set a speed 5 or greater and download again.


The LaserCut layer setting process is not too user friendly. Be patient, ask questions if you need help.

Set Job Origin[edit]

  • On the top menu bar to to the laser menu and select Set laser origin
  • Verify the origin is set to left-down, unless needed otherwise


Saving Files[edit]

The native .ECP file format saves lines and color information but not order or power settings. However, the name associated .INI file contains cut and power setting information and the .G00 file contains engraving information. [This info needs to be verified.]

If Laser Machine Is Off[edit]

  • Turn chiller and exhaust fan on
  • Turn laser cutter on (above machine control panel, next to red Stop button, turn key left-to-right)

Download file to Laser[edit]

  • In the far right-hand Layer pane
    • Click the Download button (lower right)
    • In the popup window that appears
      • Click the Del all button (delete all)
      • Click the Download current button
If during download processing the software posts a found overlap message, there are overlapping duplicate vectors in the file. Inspect for and remove duplicate vectors.

Job Setup[edit]

  • Place material on bed
  • Adjust materials bed. Set top of material approximately even with the bed rim.
Material top sticking up above the bed rim, combined with operator inattention, can and has led to laser head assembly misalignment and damage. Just ask me (Steve).
 
Beware: On the large laser the honeycomb tray is a tight fit within the bed. Check that the tray is clear of the rim. If the tray catches under the rim bad things happen.
  • Bed size is 18x24 inches for the small laser
If cutting thin material -- like paper or lightweight cardboard -- place a board or other stiff material underneath so the probe has sufficient resistance to properly calibrate Z.
 
The honeycomb bed is magnetic.

Laser Alignment Dot[edit]

On the large laser machine (BFL), the red alignment dot does not fall directly below the laser head until Z datum is set.

Set Material Origin[edit]

  1. Check Menu, if LASER (or job name) is highlighted press Esc to escape from menu mode
  2. Use arrow keys around Z to position laser head at material origin
    1. Before it cuts the machine will align the material and software origin points
  3. Press enter -- machine automatically selects the currently loaded file (for an unnamed file the menu shows LASER)
  4. Press enter -- enters Set Logic Org (set logical origin)
  5. -- If on small laser machine, machine automatically selects Yes, press enter

Set Z[edit]

  1. Press Esc to escape menu mode
  2. Move probe and laser head assembly away from bed rim. Make sure laser head and probe are completely over your material and away from the edge before setting Z
  3. Press Z to go into Z-Mode -- Menu shows Z-Axis Operate
  4. Press Datum to set Z -- table raises to calibrate probe
  5. -- If on small laser machine, Press Z to get out of Z-Mode
  6. Press Test to check cut area stays within the bounds of the material
  7. If cut area checks out, press Start/Stop

  • 1: Press Esc

  • 2: Use arrow keys to position laser head

  • 3: Press Z

  • 4: Press Datum

  • 5: Press Z

Test Cut Area[edit]

  1. Press Test to check that the full cut area stays within the bounds of the material

  • 1: Press Test

If during the test the machine posts a Soft Stop error, the most likely cause is one or more stray vectors that lie unseen outside the job bounding box. In LaserCut, zoom out, look for and delete stray vectors. Also, do the same in the original import file, typically an AI file.

Start[edit]

  1. If cut area checks out, press Start/Pause

  • 2: Press Start/Pause

  • Check that top of material is even with bed rim
  • Check laser head is over material before setting the origin
  • Check both probe and laser are over material and well away from the bed rim before setting Z
  • Run Test to check that laser head remains over the material the entire run

Making LaserCut Compatible Illustrator Files[edit]

Use the LaserCut Color Palette[edit]

  • In AI go to the Swatch pane and then open the drop-down menu in the upper-right corner of the pane (just below the X)
  • Select Open Swatch Library
  • Select LaserCut -- this will load the LaserCut color palette
    • A LaserCut color applied in AI, exported as DXF version R14/LT98/LT97 file, will import with applied colors in LaserCut 5.3

Join Disconnected Paths into Continuous Paths[edit]

  • Use the direct select tool to select all paths you want to make into a continuous path, then go to the menu Object --> Path --> Join.

Transform Font Text into Vectors[edit]

  • If the text is in font form
    • Right click the text element and select outline. This creates a vector outline of the text and thereby makes it laser software readable.

Layer Order in Illustrator[edit]

  • Layer order in Illustrator (starting with the bottom most layer) determines the path run order in LaserCut.
    • For example, if you have one path per layer in Illustrator, regardless of on-screen layout, the layer order from the bottom up will determine path cut order when imported in LaserCut.
    • To reorder paths in LaserCut use the Setting Output Sequence list box widget under the Tool drop-down menu.

Use DXF as Intermediate Format to Preserve Color Information When Moving from AI to LaserCut[edit]

  • Use the LaserCut color palette in AI to conform to the LaserCut color palette
  • Export AI file as a DXF R14/LT98/LT97 or 2000/LT200 (this DXF format retains the color palette information and imports in LaserCut)
  • In LaserCut set layer properties -- speed, power, etc.
  • Save LaserCut file in the native ECP format.
  • To import a new DXF, open the matching LaserCut ECP project file -- this brings in all previous layer information.
  • Delete existing vectors.
  • Import new DXF -- the import process will map all colors in the DXF file onto the pre-existing but hidden layer information inherited through the ECP file open process.

Save as Illustrator 10 (this does not retain color info for import)[edit]

  • To prep file for laser software, save vector file as an Illustrator 10 file.
  • If checked, un-check the Use Compression tick box

AI Scripts[edit]

Ordering pathItems for in AI to set vector run order in LaserCut[edit]

For imported AI files, the layer and z-order of paths in Illustrator determines the cut order in LaserCut.

The following script reorders paths (and paths only) within each AI layer, left to right, top to bottom.

For best fit results (though not necessary), resize artboard to the size of the contained art work. To do this go to the Object menu --> Artboard --> Fit to Artwork Bounds. 

<!--
 For each layer, order the pathItems so laser will have less travel time that if randomly placed
-->

if (app.documents.length > 0 && app.activeDocument.pathItems.length > 0 && app.activeDocument.layers.length >0 ) {
	
	// Get the active document
	var doc = app.activeDocument;
	
	// Get the active artboard dimensions
	var artIdx = doc.artboards.getActiveArtboardIndex();
	var artBoard = doc.artboards[artIdx];
	var activeRect = artBoard.artboardRect;
	
	// Get origin (top left) of artboard
	var x0 = activeRect[0];
	var y0 = activeRect[1];
	
	// Get Size of artboard
	var rectLength = Math.ceil(activeRect[2] - activeRect[0]);
	var rectHeight = Math.abs(Math.ceil(activeRect[3] - activeRect[1]));
	
	// Figure out a unitStep (divide the short side by a num and use that as unit length)
	// make a grid with gridNum buckets along the short side of the x, y plane 
	var gridNum = 25; 
	var xStep = Math.round(rectLength/gridNum);
	var yStep = Math.round(rectHeight/gridNum);
	var unitStep = xStep > yStep? yStep : xStep;
	
	// Determine matrix size
	var numXSteps = Math.floor(rectLength/unitStep);
	var numYSteps = Math.floor(rectHeight/unitStep);
	
	// create matrix
	var matrix = new Array();
	for(var s=0; s < numYSteps; s++) {
		matrix[s] = [];
		for( var t=0; t < numXSteps; t++) {
			matrix[s][t] = [];
		}
	}

	// iterate pathItems and place in proper matrix cell (bucket)
		var iX, iY
		for(var i=0; i < doc.pathItems.length; i++) {
			iX = Math.floor((doc.pathItems[i].left - x0)/unitStep);
			iY = Math.floor(Math.abs(doc.pathItems[i].top - y0)/unitStep);
			if(iX > -1 && iY > -1) {
				matrix[iY][iX].push(doc.pathItems[i]);
			}
		}
	
	// unwind matrix and set zOrder for pathItems
	for( var col = 0; col < matrix.length; col++) {
		for(var row = 0; row < matrix[col].length; row++) {
			if(matrix[col][row].length) {
				for(var z = matrix[col][row].length; 0 < z; z--) {
					matrix[col][row][z - 1].zOrder(ZOrderMethod.BRINGTOFRONT);
					//alert(matrix[col][row][z - 1]);	
				}
			}
		}
	}
	alert('The path item reorder process complete.');
	matrix = null;
}
